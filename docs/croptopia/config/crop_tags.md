# Crop Tags (1.18.2+)

In versions of Croptopia for 1.18.2 and onwards, biome datapack tags are used to configure
where crops can spawn.

**Can be modified in _data/croptopia/tags/worldgen/biome/has_crop_**

Mod Devs: can add your biomes with croptopia:has_crop/
Users: can do the same thing, or make a PR/issue with the modded biomes you want to see certain crops added to
Modpack Devs: Custom datapack with the above structure, or PR/issue with the modded biomes you want to see crops added to.
